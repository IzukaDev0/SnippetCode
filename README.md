# WhatsApp Bot - System Monitor Snippet

Kumpulan snippet kode untuk WhatsApp bot dengan fitur System Monitoring menggunakan Canvas.

## 📁 File Snippets
- `system-monitor.snippet.json` - Format JSON untuk code snippet
- `system-monitor-snippet.md` - Format Markdown dengan penjelasan

## 🚀 Fitur
- Monitoring CPU, Memory, Storage
- Tampilan gauge chart
- Informasi sistem real-time
- Memory usage Node.js

## 📦 Dependencies
```bash
npm install canvas
